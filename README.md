# 🍅 番茄工作法计时器

一个美观、功能完整的番茄工作法计时器，帮助您提高工作效率。

## ✨ 功能特性

### 🕒 核心功能
- **智能计时器**：25分钟工作 + 5分钟休息的经典番茄工作法
- **多种模式**：工作模式、短休息、长休息
- **进度可视化**：实时进度条显示剩余时间
- **声音提醒**：时间到时有声音提示
- **自动切换**：可设置自动开始下一个阶段

### 📊 统计功能
- 完成番茄数统计
- 总专注时间统计
- 今日番茄数统计
- 数据本地保存

### 📝 任务管理
- 添加/删除任务
- 标记任务完成
- 任务列表持久化存储

### ⚙️ 个性化设置
- 自定义工作时间
- 自定义休息时间
- 开关声音提醒
- 开关自动切换

## 🚀 快速开始

### 方法1：直接打开
1. 双击 `index.html` 文件
2. 在浏览器中打开使用

### 方法2：使用本地服务器
```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后在浏览器中访问：`http://localhost:8000`

## 🎯 使用指南

### 基本使用
1. **选择模式**：点击工作、短休息或长休息按钮
2. **开始计时**：点击"开始"按钮
3. **专注工作**：避免干扰，专注当前任务
4. **休息调整**：时间到后休息，准备下一个番茄钟

### 任务管理
1. **添加任务**：在输入框中输入任务，点击"添加"
2. **完成任务**：勾选任务前的复选框
3. **删除任务**：点击任务右侧的删除按钮

### 个性化设置
1. **调整时间**：在设置中修改各阶段时长
2. **开关功能**：开启/关闭声音提醒和自动切换
3. **保存设置**：点击"保存设置"按钮
